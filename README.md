# Image Gallery

This is an Image Gallery project made using HTML, CSS and JavaScript.

## Features

- View images in a gallery
- Preview images in a larger view
- Next and Previous buttons
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone the project.
2. Open the `index.html` file in your browser.
3. Browse the image gallery.

- JavaScript DOM manipulation
- Creating interactive user interfaces
